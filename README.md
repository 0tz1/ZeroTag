# Artwork Detail Card (React)

Single-card React UI showcasing an artwork detail view with speech synthesis narration.

## Getting started

```bash
npm install
npm run dev
```

Open the local URL printed in the terminal.

## Build for production

```bash
npm run build
npm run preview
```

## Notes

- The "Play Audio" button uses the browser Web Speech API. Unsupported browsers display a fallback message.
- All artwork data is hardcoded in `src/App.jsx`.
